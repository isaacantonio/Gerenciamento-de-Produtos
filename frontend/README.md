Pré-requisitos
Node.js e npm

Execute o comando abaixo para instalar as dependências do front-end:
npm install

Para rodar a aplicação localmente em modo de desenvolvimento, execute:
npm run dev

A aplicação estará disponível em http://localhost:3000.
